# Project 9: Automated AWS IAM Security Auditor

## Objective
To automate the auditing of Identity and Access Management (IAM) policies in an AWS environment, specifically targeting the enforcement of Multi-Factor Authentication (MFA) across all user accounts.

## Skills Demonstrated
* **Cloud Infrastructure:** Configured AWS CLI and managed IAM user permissions safely using dedicated lab environments.
* **Python for Cloud (Boto3):** Utilized the official AWS SDK to interact with cloud APIs programmatically.
* **Compliance Enforcement:** Automated the detection of policy violations to maintain strict access control standards.

## Manager's Summary
In cloud environments, compromised credentials are the fastest vector for a data breach. 
Manually checking dozens or hundreds of AWS accounts for MFA compliance is highly inefficient and prone to human error. 
By leveraging Python and `boto3`, I built an automated auditing tool that instantly flags non-compliant accounts, allowing security teams to enforce Zero Trust principles at scale.
