# Employee Security Quick-Reference Guide

## 1. Passwords are WEAK. Use Passphrases.
* **Bad:** `Spring2026!` (Hackers crack this in 2 seconds).
* **Good:** `BlueCoffeeMugRunningFast!` (This takes centuries to crack).
* **The Rule:** Length beats complexity. String 4 random words together.

## 2. Multi-Factor Authentication (MFA)
* Think of your password as the key to the front door, and MFA as the security guard asking to see your ID. 
* Always use an Authenticator App (I use Google Authenticator) instead of SMS text messages, as text messages can be intercepted.

## 3. The "Hover" Rule for Phishing
* Never trust the name on an email! Hackers fake the "Sender Name" easily.
* **The Fix:** Hover your mouse over the link before clicking.
* If the email is from Microsoft, but the link goes to `http://login-update-xyz.com`, report it immediately.
* I copy and paste links instead of clicking them so I can read the URL. If it's wrong, I am not clicking "enter".
